# Personal & Market Finance Intelligence

A beginner Python project focused on personal finance tracking and market data analysis.  
This is the first project in my FinTech journey at the University of Sussex.

## Project Goal
- Build strong foundations in data processing and financial analysis
- Understand the difference between **personal data** and **market data**
- Practice basic data visualization and simple prediction

## Main Features
- Record and manage income/expenses
- Analyze spending patterns by category and month
- Simple spending forecasting (regression model)
- Fetch and analyze stock/crypto market data using yfinance
- Visualize data with charts

## Technologies Used
- Python
- pandas (data manipulation)
- matplotlib & seaborn (visualization)
- yfinance (stock and crypto data)
- Jupyter Notebook

## Project Structure
```
personal-market-finance-intelligence/
- data/                # raw and processed datasets
- notebooks/           # Jupyter notebooks for analysis and experiments
- src/                 # core Python scripts (data processing, models)
- app.py               # main script to run analysis
- requirements.txt     # dependencies
- README.md            # project documentation
```
